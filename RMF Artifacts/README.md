# RMF Artifacts Index

This folder contains all required artifacts for the NIST Risk Management Framework (RMF) package.

## Included Artifacts

1. **Security Categorization Report (SCA)**  
   Documents the system categorization following FIPS 199 and NIST SP 800-60.

2. **Control Implementation Summary (CIS)**  
   Summarizes how each NIST 800-53 control is implemented within the system.

3. **Security Assessment Plan (SAP)**  
   Defines the assessment approach, methods, and procedures for evaluating security controls.

4. **Continuous Monitoring Strategy**  
   Outlines the organization’s approach to ongoing assessments, tracking, and reporting.

5. **Boundary Diagram**  
   Visual depiction of the system boundary, components, interconnections, and trust zones.

6. **Data Flow Diagram (DFD)**  
   Illustrates the movement and processing of data across system components.

7. **Asset Inventory Template**  
   Provides a structured list of hardware, software, cloud resources, and system components.

8. **Control Traceability Matrix (CTM)**  
   Maps each control to its corresponding implementation, evidence, and responsible party.

---

## Purpose

This repository is structured to support RMF Phase 1–3 activities and prepare for security assessment and authorization.  
All artifacts in this folder are formatted for audit readiness and easy integration into SSP, SAR, and POA&M workflows.

- Policies

Each artifact is provided in Markdown format to ensure compatibility with GitHub, GitLab, and documentation pipelines.
