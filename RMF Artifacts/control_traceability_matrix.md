# Control Traceability Matrix (CTM)

**File:** control_traceability_matrix.md  
**Category:** RMF Artifacts  
**Standard:** NIST SP 800-53 Rev 5

---

## 1. Purpose
The CTM maps security controls to policies, procedures, evidence, and responsible parties to ensure complete traceability.

---

## 2. Traceability Table
| Control | Policy Reference | Procedure Reference | Evidence | Responsible Role |
|--------|------------------|---------------------|----------|------------------|
| AC-2 | Access Control Policy | User Access Request Procedure | Access logs, request forms | IAM Lead |
| IR-4 | Incident Response Policy | Incident Response Procedure | Tickets, reports | SOC Manager |
| CM-3 | Configuration Mgmt Policy | Change Mgmt Procedure | Change tickets | Change Manager |
| RA-3 | Risk Assessment Policy | Risk Methodology Procedure | Risk report | ISSO |
