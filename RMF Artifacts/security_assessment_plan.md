# Security Assessment Plan (SAP)

**File:** security_assessment_plan.md  
**Category:** RMF Artifacts  
**Role:** Security Control Assessor (SCA)

---

## 1. Purpose
The SAP outlines the assessment approach, methodologies, and schedule to evaluate security controls for the information system.

---

## 2. Assessment Scope
- System boundaries
- Interfaces with external systems
- Components in-scope for testing

---

## 3. Assessment Objectives
Objectives include:
- Validate effectiveness of implemented controls  
- Identify vulnerabilities and gaps  
- Provide risk-based recommendations

---

## 4. Assessment Methods (per NIST SP 800-53A)
| Method | Description |
|--------|-------------|
| **Examine** | Review policies, procedures, diagrams, configurations. |
| **Interview** | Question personnel on responsibilities and activities. |
| **Test** | Execute technical tests to confirm control functionality.|

---

## 5. Schedule
Provide estimated dates for:
- Kickoff  
- Testing window  
- Evidence requests  
- Draft SAR delivery  
- Final SAR approval

---

- Lead Assessor  
- Technical Tester  
- Documentation Reviewer
