# Audit Logging & Monitoring Checklist

> Use this checklist to ensure proper logging, monitoring, and alerting of system activity.

---

## 1. Log Collection
- [ ] Logs collected from all key systems  
- [ ] Logs centralized to SIEM or log server  
- [ ] Time synchronization (NTP) enabled  

---

## 2. Log Content
- [ ] Admin activity logged  
- [ ] Authentication events logged  
- [ ] System changes logged  
- [ ] Network security events logged  

---

## 3. Monitoring
- [ ] Alert thresholds configured  
- [ ] Daily log reviews performed  
- [ ] Suspicious activity escalated  
- [ ] Automated alerting functioning  

---

## 4. Retention
- [ ] Logs retained per policy (e.g., 1 year or more)  
- [ ] Logs tamper-protected  
- [ ] Backups of log archives maintained  

---

## 5. Documentation
- [ ] Logging diagrams maintained  
- [ ] Monitoring procedures updated  
- [ ] Exceptions documented
