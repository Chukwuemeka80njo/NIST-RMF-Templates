# Access Control Checklist

> Use this checklist to maintain strong user access and account control.

---

## 1. Account Management
- [ ] All accounts approved by management  
- [ ] Unique user IDs assigned  
- [ ] No shared accounts in use  
- [ ] Orphaned accounts reviewed monthly  
- [ ] Terminated user accounts disabled within 24 hours  

---

## 2. Authentication
- [ ] MFA enabled for all privileged accounts  
- [ ] MFA enabled for remote access  
- [ ] Password policy enforced  
- [ ] Default passwords removed  

---

## 3. Privilege Management
- [ ] Least privilege enforced  
- [ ] Privileged access reviewed quarterly  
- [ ] Admin accounts separated from standard accounts  
- [ ] Elevated access removed when no longer required  

---

## 4. System Access
- [ ] Role-based access controls (RBAC) applied  
- [ ] Access to sensitive systems logged  
- [ ] Remote access controlled and monitored  

---

## 5. Reviews & Audits
- [ ] Monthly user access review completed  
- [ ] Quarterly privileged access audit completed  
- [ ] Access documentation updated
