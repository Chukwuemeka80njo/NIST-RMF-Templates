# Configuration Management Checklist

> Use this checklist to ensure systems are configured securely and consistently.

---

## 1. Baseline Configuration
- [ ] Baseline configurations documented  
- [ ] Baselines approved by security  
- [ ] Deviations documented and approved  

---

## 2. Change Control
- [ ] Change requests submitted and logged  
- [ ] Risk analysis performed for each change  
- [ ] Approvals obtained before implementation  
- [ ] Emergency changes documented  

---

## 3. Secure Configuration
- [ ] Unnecessary services disabled  
- [ ] Default accounts removed  
- [ ] Secure configuration standards applied  
- [ ] System hardened using CIS/NIST benchmarks  

---

## 4. Patch Management
- [ ] OS patches applied regularly  
- [ ] Application patches applied  
- [ ] Third-party updates tracked  
- [ ] Failed patches documented  

---

## 5. Verification
- [ ] Configuration scans executed  
- [ ] Non-compliance items remediated  
- [ ] Configuration documentation updated
