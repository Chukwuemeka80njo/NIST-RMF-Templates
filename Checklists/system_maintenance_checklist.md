# System Maintenance Checklist

> Use this checklist to ensure secure maintenance of systems.

---

## 1. Scheduled Maintenance
- [ ] Maintenance windows defined and approved  
- [ ] Maintenance activities logged  
- [ ] Overdue maintenance tracked  

---

## 2. Tools & Processes
- [ ] Only authorized tools used  
- [ ] Remote maintenance authenticated and encrypted  
- [ ] Temporary maintenance accounts removed afterward  

---

## 3. Documentation & Review
- [ ] Maintenance procedures documented  
- [ ] Post-maintenance testing completed  
- [ ] Service disruptions reviewed
